== Description ==

It will disable all unnecessary WordPress features and speed up your website in an elegant way.

By using this All-in-One plugin you can simply disable feature that you do not want to use with one click.


== Installation ==

Login to your WordPress Admin panel and search "Disable Features" in "Plugins > Add New" screen. You can install this plugin by using the following steps also:

1. Download the plugin via WordPress.org
2. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
3. Activate the plugin through the 'Plugins' menu in WordPress

Once activated go to:

4. Settings > Disable Features page
5. In Options page disable the feature that you do not want to use.
6. Once all done, click on Save Changes button to disable the features.
